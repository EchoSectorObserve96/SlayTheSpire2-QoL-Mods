## 📌 About

Slay the Spire 2 is finally here! This repository provides quality of life improvements for the game, focusing on faster animations, smoother gameplay, and UI enhancements. All modifications are safe, non-intrusive, and designed to respect the original game balance.

What it includes:  
✅ Fast Animations – speed up card plays, enemy turns, and transitions
